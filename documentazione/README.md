# TooMatch - Documentazione Progetto

## Panoramica
TooMatch è un'applicazione React Native per la gestione di match di tennis. Permette agli utenti di registrare e visualizzare i punteggi e i dettagli delle partite.

## Tech Stack
- **Frontend**: React Native (Expo)
- **Backend/Database**: Firebase Firestore

## Funzionalità Attuali
- **Visualizzazione Match**: Elenco cronologico delle partite salvate.
- **Creazione Match**: Funzionalità di test per creare un match con dati predefiniti.

## Struttura del Codice
- `App.js`: Entry point dell'applicazione. Gestisce la navigazione principale e la vista della lista match.
- `src/services/matchService.js`: Servizio per interagire con Firebase (leggi/scrivi match).
